# Contributing to Electronics Lab Guide

Thanks for your interest in contributing to this electronics lab guide.

## How to Contribute

1. **Fork** the repository
2. **Create** a new branch (`git checkout -b feature/your-idea`)
3. **Commit** your changes (`git commit -m 'Add: your feature'`)
4. **Push** to the branch (`git push origin feature/your-idea`)
5. **Open** a Pull Request

## Guidelines

- Keep content beginner-friendly
- Use clear, simple language — avoid unnecessary jargon
- Include sources and links where relevant
- Test any code or instructions before submitting

## Ideas Welcome

- Tool recommendations and reviews
- Regional buying guides (EU, US, Asia, etc.)
- Translations
- Typo and grammar fixes
- New project tutorials
- Electronics lab setup tips
- Electronics workshop and workstation tips
- Soldering technique improvements

## Content Style

- Write in plain English
- Use "you" to address the reader
- Be practical — focus on what actually works
- Include costs where relevant
- Add images if they help explain something

## Reporting Issues

If you find something confusing, incorrect, or outdated:

1. Open an issue
2. Describe what is wrong or unclear
3. Suggest a fix if you have one

Questions? Open an issue and ask.
