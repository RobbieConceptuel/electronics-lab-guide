## Upgrade Path

Do not buy everything at once. Start small and add as you need:

**Month 1:** Starter Setup - Learn the basics

**Month 3:** Add hot air station and flux - Start SMD work

**Month 6:** Upgrade soldering station, add microscope - Better quality work

**Later:** Add power supply, oscilloscope, etc as specific projects require them
