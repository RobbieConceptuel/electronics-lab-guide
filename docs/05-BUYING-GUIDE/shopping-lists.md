# Shopping Lists

Copy and paste lists for different needs.

---

## Absolute Minimum

The bare minimum to start soldering:

```
[ ] Soldering iron with temperature control
[ ] Solder wire 60/40 0.8mm
[ ] Brass tip cleaner
[ ] Something to practice on
```
